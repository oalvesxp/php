# PHP

Para usar estes arquivos, você necessida te ter o php-cli instalado na versão >= 5

```
sudo apt update 
sudo apt upgrade -y
sudo apt install php-cli -y
```
</br>
Para rodar os scripts php utilize o comando:

```
php <nome-do-arquivo>.php
```
